# BigData_analysis_of_transport_in_Ukraine_SHI33

![Обраний датасет](https://data.gov.ua/dataset/06779371-308f-42d7-895e-5a39833375f0) - Реєстр зареєстрованих тз Україна


1. Ознаки у датасеті
Датасет містить 20 основних ознак, що описують реєстраційні дії та характеристики транспортних засобів (ТЗ):
+1


PERSON: Тип особи (фізична "P" або юридична "J").
+1


REG_ADDR_KOATUU: Код адреси реєстрації за КОАТУУ.
+1


OPER_CODE: Код операції (наприклад, 315 — перереєстрація, 100 — первинна реєстрація).
+1


OPER_NAME: Опис реєстраційної дії.
+1


D_REG: Дата реєстрації.
+1


DEP_CODE: Код сервісного центру (ТСЦ).
+1


DEP: Назва сервісного центру.
+1


BRAND: Марка автомобіля.
+1


MODEL: Модель транспортного засобу.
+1


VIN: Ідентифікаційний номер ТЗ (прихований або частковий у вибірці).
+1


MAKE_YEAR: Рік випуску.
+1


COLOR: Колір ТЗ.
+1


KIND: Тип ТЗ (легковий, вантажний, мотоцикл тощо).
+1


BODY: Тип кузова (універсал, седан, хетчбек тощо).
+1


PURPOSE: Призначення (загальний, спеціалізований).
+1


FUEL: Тип пального (бензин, дизель, електро, газ тощо).
+1


CAPACITY: Об'єм двигуна.
+1


OWN_WEIGHT: Маса ТЗ без навантаження.
+1


TOTAL_WEIGHT: Повна дозволена маса ТЗ.
+1


N_REG_NEW: Новий державний номерний знак.
+1

2. Обсяг датасету

Файлова структура: Датасет представлений у форматі CSV.
+1


Оцінка обсягу: Хоча точна кількість рядків залежить від повного розпакування, виходячи з наданих фрагментів, файл містить дані про реєстрації за січень 2026 року (з 01.01.26 по 30.01.26). У фрагментах зафіксовано понад 100 000 текстових токенів лише у вибірці. Це свідчить про те, що повний файл може містити сотні тисяч записів, що робить його придатним для Big Data аналізу.
+1

3. Завдання для Big Data аналізу
Завдяки різноманітності ознак, цей датасет можна використовувати для вирішення таких завдань:

Прогнозування попиту на авторинку: Аналіз популярності марок (BRAND) та моделей (MODEL) залежно від року випуску (MAKE_YEAR) та регіону (REG_ADDR_KOATUU).

Моніторинг екологічних трендів: Відстеження динаміки переходу на електромобілі (FUEL: ЕЛЕКТРО) порівняно з традиційним пальним.

Геопросторовий аналіз: Визначення концентрації певних видів транспорту в різних областях України за кодами КОАТУУ та сервісними центрами (DEP).

Кластеризація автопарку: Групування транспортних засобів за технічними характеристиками (маса, об'єм двигуна, тип кузова) для сегментації ринку.

Аналіз реєстраційної активності: Виявлення піків навантаження на сервісні центри та найпопулярніших типів операцій (OPER_NAME), наприклад, частка перепродажів порівняно з первинною реєстрацією ввезених авто.

Виявлення аномалій: Пошук нетипових реєстрацій (наприклад, реєстрація дуже старих авто або невідповідність маси ТЗ його типу) для цілей безпеки або аудиту.
